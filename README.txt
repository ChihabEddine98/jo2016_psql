  2- IMPORTER LE FICHIER 'CREATION.sql' QUI SE TROUVE DANS LE RÉPERTOIRE 'sql/'

    -> \i sql/CREATION.sql

  3- IMPORTER LE FICHIER 'INSERTIONS.sql' QUI SE TROUVE DANS LE RÉPERTOIRE 'sql/'
    DANS VOTRE BASE DE DONNEE CREE PRECEDEMENT.

    -> \i sql/INSERTIONS.sql

LES TABLES SONT CRÉES ET PEUPLÉES ! YOU'RE GOOD TO GO !
